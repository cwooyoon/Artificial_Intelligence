## Generals
* 행렬은 사상(map)이다.
  - m x n 행렬 A에는 n차원 공간에서 m차원 공간으로의 mapping이다.
    - n 차원 공간의 점 x(n차원의 종벡터)를 m차원 공간의 점(m차원 종벡터) Ax 로 옮기는 사상이다.

* 우리는 3차원 공간에 살고있다.
  - 3차원 세계의 일을 다루기 위해서는 '공간'을 잘 기술할 수 있는 용어가 필요하다.
  - 벡터공간: 현실 공간의 성질을 특정 수준에서 추산화한 것
    - 3차원 공간 --> 2차원 평면
